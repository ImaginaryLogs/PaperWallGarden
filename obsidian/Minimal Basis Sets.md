Assigns exactly one mathematical function per native atomic orbital. A minimal basis set is where only sufficient orbitals are used to contain all the electrons in the neutral atom.

# STO-nG basis sets
The basis functions are linear combinations of $n$ primitive [Gaussian-type orbitals](https://en.wikipedia.org/wiki/Gaussian_orbital "Gaussian orbital") (GTOs) that are fitted to single [Slater-type orbitals](https://en.wikipedia.org/wiki/Slater-type_orbital "Slater-type orbital") (STOs).
